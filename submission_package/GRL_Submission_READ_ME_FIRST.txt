--- READ ME FIRST: HOW TO EXTRACT THE CODEBASE ---

To comply with Moodle's 50MB file limit, the project codebase has been split into multiple volumes.

INSTRUCTIONS:
1. Download ALL 3 submission files into the SAME folder on your computer:
   - GRL_Portfolio_Submission.z01
   - GRL_Portfolio_Submission.z02
   - GRL_Portfolio_Submission.zip

2. To extract, double-click ONLY the file named:
   GRL_Portfolio_Submission.zip

3. You will automatically combine the volumes and extract the full "graph-rl-portfolio" directory.

NOTE: 
The .z01 and .z02 files are split volumes and cannot be opened individually. They must be present in the same directory as the .zip file for extraction to succeed.

Once extracted, please refer to the README.md in the root directory for setup and reproduction instructions.
